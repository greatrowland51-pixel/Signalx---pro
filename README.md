# SignalX Pro v3.1

A live Deriv trading dashboard. Auth is Deriv-API-token only. Connects
directly to Deriv's WebSocket API from the browser (no backend needed —
Deriv's API supports direct browser connections).

## What's real vs. simulated right now

- **Real**: connecting to your Deriv account, reading your live balance,
  pulling live tick/candle history, live indicator math (EMA/RSI/MACD/ADX/
  Bollinger) computed on those live prices.
- **Simulated by default**: trade *execution*. Settings → "Enable Live
  Trading" is OFF out of the box. While off, the app shows you what bots
  *would* have done against live prices without spending real money.
  Turning it on wires `buyContract()` in `src/lib/useDeriv.js` to place
  real contracts with real money.

**Do not flip Live Trading on until you've watched the simulated numbers
for a real stretch of time and you understand exactly what each bot does.**

## Before going live for real

1. **Get your own Deriv `app_id`** at https://developers.deriv.com (free,
   takes 2 minutes) and replace `APP_ID = 1089` in `src/lib/useDeriv.js`.
   `1089` is Deriv's shared demo app — fine for testing, not for
   production use with real users.
2. **Wire `buyContract` into the trade-resolution loop** in `src/App.jsx`
   (search for the comment "resolves the trade via simulation"). Right
   now that block always simulates win/loss instead of calling
   `deriv.buyContract(...)`. This is intentional — flipping it requires
   you to also handle contract confirmation, payout polling via
   `proposal_open_contract`, and error handling for rejected trades.
3. **Test on a Deriv demo account first** (tokens starting `VRTC` are
   demo accounts — the app shows a "DEMO" badge automatically when
   connected to one).

## Deploy to Vercel

```bash
npm install -g vercel   # one-time
cd signalx-app
vercel
```
Follow the prompts (link/create a project, accept defaults — it auto-
detects Vite). It'll give you a live `.vercel.app` URL in under a minute.
For production: `vercel --prod`.

## Deploy to Netlify

```bash
npm install -g netlify-cli   # one-time
cd signalx-app
npm run build
netlify deploy --dir=dist   # preview
netlify deploy --dir=dist --prod   # production
```

Or drag-and-drop the `dist/` folder (after `npm run build`) onto
https://app.netlify.com/drop — no CLI needed.

## Local development

```bash
npm install
npm run dev
```
Opens at `http://localhost:5173`.


## On the digit bots

Deriv's synthetic-index digit outcomes are generated by a fixed-
probability RNG — each tick is statistically independent of the last.
No amount of pattern-matching (Markov chains, streak-counting, chi-
square scans) can predict the next digit from past digits, and any tool
claiming otherwise is misrepresenting what it's doing. The 6 digit-market
"bots" here are built around what's actually measurable: whether the
generator's live output frequency ever drifts outside its expected
statistical band, and whether Deriv's payout-implied odds ever
mismatch the live empirical probability. Both are honest, real things
to monitor — neither predicts the next tick.
